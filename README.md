# مشروع: رعاية

## نبذة عن المشروع

**رعاية** هو تطبيق صحي يهدف إلى مساعدة مرضى الضغط والسكر على متابعة حالتهم الصحية بشكل يومي بأسلوب بسيط وسهل الاستخدام. يساعد التطبيق المستخدمين على تنظيم نظامهم الغذائي، متابعة مؤشراتهم الصحية، والحصول على نصائح لنمط حياة صحي، مما يعزز الالتزام بالعادات الصحية دون تعقيد. كما يسعى إلى تقديم دعم جسدي ونفسي ليكون رفيقًا يوميًا يمنح المستخدم شعورًا بالطمأنينة والاستمرارية في الاهتمام بصحته.

## أعضاء الفريق والمسؤوليات

| اسم العضو             | المسؤولية                                                                                                                                         |
| --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------- |
| **محمد علاء المصلحي** | **Team Leader** – إدارة المشروع، توزيع المهام، متابعة سير العمل، مراجعة التصميمات، وتجميع النسخة النهائية للمشروع.                                |
| **حسام عادل طايل**    | **UX Researcher** – إجراء بحث المستخدم، تحليل المنافسين، تحديد الفئة المستهدفة، إعداد Personas وEmpathy Map، وتحديد متطلبات المستخدم.             |
| **محمد ناصر شرف**     | **UX Designer** – تصميم Information Architecture، User Flow، User Journey، وإنشاء Wireframes منخفضة ومتوسطة الدقة.                                |
| **مريم يحيى حامد**    | **UI Designer** – تصميم الواجهات النهائية (High-Fidelity UI)، إنشاء Design System، اختيار الألوان والخطوط، وتصميم المكونات البصرية.               |
| **هاجر نصر محمد**     | **Prototype & Usability Testing** – إنشاء Prototype تفاعلي باستخدام Figma، إجراء اختبارات قابلية الاستخدام، توثيق النتائج، وإعداد العرض التقديمي. |
