اسم المشروع : رعاية

وصف المشروع : رعاية هو تطبيق صحي يهدف إلى مساعدة مرضى الضغط والسكر على متابعة حالتهم الصحية بشكل يومي وبأسلوب بسيط وسهل الاستخدام. يساعد على تنظيم الاكل ويوفّر نصائح لنمط حياة صحي تساعد المستخدم على الالتزام دون تعقيد. يسعى التطبيق إلى دعم المستخدم جسديًا ونفسيًا، ليكون رفيقًا يوميًا يساعده على الاهتمام بصحته بثبات وطمأنينة.
